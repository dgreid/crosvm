Expand description

Nested Target Extension - Set/Remove/Reset the Environment variables for spawned processes.

Corresponds to GDB’s set environment cmd.

Note: Environment variables are not guaranteed to be UTF-8, and are passed as raw byte arrays. If the provided keys/values could not be converted into an appropriate representation, a non fatal error should be returned.
