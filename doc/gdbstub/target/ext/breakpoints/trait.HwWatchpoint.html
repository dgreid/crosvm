Expand description

Nested Target Extension - Set/Remove Hardware Watchpoints.

See the GDB documentation regarding watchpoints for how they’re supposed to work.

Note: If this extension isn’t implemented, GDB will default to using software watchpoints, which tend to be excruciatingly slow (as hey are implemented by single-stepping the system, and reading the watched memory location after each step).
