This module implements the “Vhost User” Virtio device as specified here - https://stefanha.github.io/virtio/vhost-user-slave.html#x1-2830007. The device implements the Virtio-Vhost-User protocol. It acts as a proxy between the Vhost-user Master (referred to as the Vhost-user sibling in this module) running in a sibling VM’s VMM and Virtio-Vhost-User Slave implementation (referred to as device backend in this module) in the device VM.
