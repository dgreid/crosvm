fn assign_irq(
    &mut self,
    _irq_evt: &Event,
    _irq_resample_evt: &Event,
    _irq_num: Option<u32>
) -> Option<(u32, PciInterruptPin)>

Assign a legacy PCI IRQ to this device. The device may write to irq_evt to trigger an interrupt. When irq_resample_evt is signaled, the device should re-assert irq_evt if necessary. Optional irq_num can be used for default INTx allocation, device can overwrite it. If legacy INTx is used, function shall return requested IRQ number and PCI INTx pin.
