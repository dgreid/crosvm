Trait cros_async::IntoAsync[−][src]

pub trait IntoAsync: AsRawFd { }
Expand description

Marker trait signifying that the implementor is suitable for use with cros_async. Examples of this include File, and sys_util::net::UnixSeqpacket.

(Note: it’d be really nice to implement a TryFrom for any implementors, and remove our factory functions. Unfortunately https://github.com/rust-lang/rust/issues/50133 makes that too painful.)
